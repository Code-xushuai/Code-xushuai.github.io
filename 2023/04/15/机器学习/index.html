  2. 数据集的构成

# 2.2.5. Tf-idf 文本特征提取

  • TF-IDF 主要思想：如果某个词或者短语在一章文章中出现的概率高，并且在其他文章出现很少，则认为此词或者短语具有良好的类比区分能力，适合用来分类
  • TF-IDF 作用：用以评估一字词对于文件集或一个语料库中的其中一份文件的重要程度

# 2.2.6. 特征预处理

  • 为什么需要进行归一化、标准化
    特征的单位或者大小相差较大，或者某特征的方差相比其他的特征要大出几个数量级，容易影响目标结果，使得算法无法学习到其他特征

  • 归一化

  1. 定义
    通过对原始数据进行变换把数据映射到（默认）[0,1] 之间

  2. 公式

  3. api

  4. 归一化处理

  1. 归一化总结
    最大值、最小值容易受异常值影响，只适用与传统精确小数据场景
  • 标准化

  1. 公式
    (x - mean 均值) /std 标准差

  2. 归一化、标准化区别
    对于归一化来说：如果出现异常点，影响最大值、最小值，那么结果显然会发生改变
    对于标准化来说：如果出现异常点，由于具有一定数据量，少量的异常点对于平均值的影响不大，从而方差改变较小

  3. 代码

  1. 应用场景
    在已有样本足够多的情况下较为稳定，适合现代嘈杂大数据

# 2.2.7. 特征降维

  1. 概念
    降维是指在某些限定条件下，降低随机变量（特征）个数，得到一组 “不相关” 主变量过程

  2. 降维方法

  • 特征选择

    1. 定义
      找出数据中含有事物共同特征的

    2. 方法

      1. Filter 过滤式
        • 方差选择法：低方差特征过滤
        • 相关系数
      2. Embedded 嵌入式
        • 决策树
        • 正则化
        • 深度学习
  • 低方差特征过滤
