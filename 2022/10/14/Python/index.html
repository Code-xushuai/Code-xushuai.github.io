      - range(num): 获取从0开始到num(但是不包含num本身)

数据容器

  • list（列表）
    • name_list=[“python”,’xs’,1,”a”,True]
      a=[[1,2,3,],[4,5,6,]]
    • 下标索引(反向)：my_list[0]
      嵌套列表：My_list[0][0]
    • 向列表增加一个元素：列表.append(元素)
    • 在指定下标插入元素：列表.insert(下标,元素)
    • 删除列表指定下标元素：列表.pop(下标)
    • 除去元素(从前往后第一个元素)：列表.remove(元素)
    • 清空列表：列表.clean
    • 查找指定下标元素：列表.index(元素)
    • 统计列表长度：len(列表)
    • 统计列表元素出现次数：列表.count(元素)
    • 打印一个含有十个0的列表
  • tuple(元组：无法修改)
    • t1=(“python”,’xs’,1,”a”,True)
    • 下标索引(反向)：t1[0]
      嵌套列表：t1[0][0]
    • 查找指定下标元素：元组.index(元素)
    • 统计元组长度：len(元组)
    • 统计元组元素出现次数：元组.count(元素)
  • str(字符串：无法修改)
    • my_name=”xushuai”
      my_name[0]
    • replace方法（将str1替换为str2）：string.replace(被替换,替换)
    • split方法（将字符串以什么字符相分割）：string.split(“分隔符”)
      得到的结果为一个列表
    • strip方法(去除前后的空格)：string.strip()
           (去除前后指定字符串): string.strip("字符串")
    • 统计字符串元素出现次数：string.count(字符串)
    • 统计字符串长度：len(字符串)
  • 序列（切片）
  • 集合set：不支持下标索引，允许修改
    • my_set={“xu”,”s”,”c”}
    • 集合.add(元素)：添加
    • 集合.remove(元素)：移除指定元素
    • 集合.pop()：随机取出一个元素
    • 集合1.difference(集合2)：差集(得到一个新集合,集合1有集合2没有)
    • 集合1.union(集合2)：合集(得到新集合)
    • 集合1.difference_update(集合2)：在集合1中删除集合2中的元素
  • 字典dict
    • my_dict={key1:value1,key2:value2,key3:value3}
    • 字典.[key]：获取指定key的value
    • 字典.[key]=value：添加或修改
    • 字典.pop(key)：取出对应的value值并在字典中删除
    • 字典.keys：获取字典中所有的key，可用于for循环
    • len(字典)：计算字典的元素
    • 字典嵌套(表格形式)
  • 容器通用排序功能：sorted(容器,[reverse=True])
    reverse: 用于反转排序，默认为False
  • 容器之间可以相互转换，字典除外

函数

  • 传入参数
    • 不定长–>位置不定长
      def 函数名(参数:*args)
      *args: args是元组类型
      例如: 函数名(“Tom”,18)–>所有参数都会被args收集，合并为一个元组，这叫位置传递
    • 不定长–>关键字不定长
      def 函数名(参数:**kwargs)
      **kwargs：传入以字典的形式传入(key=value)
      例如: 函数名(name=”Tom”,age=18,id=110)
    • 函数作为参数进行传入：计算逻辑传入
  • 定义函数
    • def关键字：定义带有名称的函数（可以重复使用）
    • lambda关键字：定义匿名函数，没有名称（只可临时使用一次）
    • –>lambda 传入参数:函数体（一行代码）
      例子：fun(lambda x,y:x+y)

文件

  • open():打开函数
    • open(name,mode,encoding)
    • name: 打开文件名的字符串(可以包含文件所在的具体路径)
    • mode：只读’r’，只写’w’，追加’a’
        w: 文件原有内容被删掉
  a: 在原有文件基础上进行追加
    • encoding：编码格式
    • 示例：f=open(“python.txt”,”r”,encoding=”UTF-8”)
  • 文件方法
    • f.read(num): num表示从文件中读取的数据长度（单位：字节），若没有传入默认读取所有的数据
      • 如果程序程序中多次调用read()方法，下一个会在上一个读取结尾处继续读取
    • f.reaflines(): 按照行的方式把整个文件内容一次性的读取，返回一个列表，其中每一行的数据为一个元素
      • 在一个程序中读取一个文件对象，无论用什么方法读取，方式都是续接（指针）
    • f.reafline(): 每次读取一行
    • f.close
    • for循环读取文件
    • with open方法：文件不用手动关闭
    • f.wrint: 写入文件
    • f.flush: 将内存中的内容写入到硬盘中（先刷新在关闭）
