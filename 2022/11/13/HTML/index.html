简介

  • HTML：超文本(流媒体、图片、视频、声音···)标记语言，由大量得标签组成，每一个标签都有开始和结束
  • html：根标签
  • head：子标签
  • title：网页标题
  • body：显示的内容

基本标签

标签 描述
< p > 段落标签
< h > 标题字—>一级< h1>, 六级< h6>)
< br > 换行标签（独目标记)
< hr > 水平线标签（独目标记)
< pre > 预留格式
< del > 删除字
< ins > 插入字
< i > 斜体字
< b > 粗体字
< font > 字体标签
< sup > 右上角加字102
< sub > 右下角加字10m

实体符号(分号结尾)


布局标签div、span

  • < div >和< span >是一个盒子，div为大盒子，span为小盒子
  • < div >大盒子，自己独占一行
  • < span >小盒子，一行可以方多个

列表

  • 无序列表：使用ul标签来创建无序列表
    < ul >中只能嵌套，直接在< ul >标签中输入其他标签或者文字的做法是不被允许的。

  • 有序列表：使用< ol >标签来创建无序列表
    使用< li >表示列表项

  • 自定义列表：使用d标签来创建一个定义列表
    使用dt来表示定义的内容
    使用dd来对内容进行解释说明


超链接

href属性：是必须写的，可以是外部链接也可以是内部链接
target属性：用于指定链接页面的打开方式，其中_self为默认值，__blank为在新窗口中打开方式.


图片 音频

  • src属性必须写
  • 相对路径和绝对路径
    • 绝对路径:是指日录下的绝对位置，百接到达E标位置，通常是从盘待开始的路径.

表单form

  • 作用：获取用户输入信息
  • 表单组成
  • form标签属性
  • 表单控键
    • < input >表单元素
      • type: text, password
      • type: radio 单选按钮

      • type: checkbox 多选复选框

      • name属性：定义input元素的名称，用于对提交到服务器后的表单数据进行标识，
      • value属性：定义input元素的默认值
      • submit属性：定义提交按钮。提交按纽会把表单数据发送到服务器。
      • button属性：普通按钮 button后期结创s搭配使用
      • file属性：上传文件
    • lable标签：增加input的点击范围
    • < select > 下拉选项
      在页面中，如果有多个选项让用户选择，并且想要节约页面空间时，我们可以使用< select >标签控件定义下拉列表。
    • < textarea >文本域
      应用余留言、建议
