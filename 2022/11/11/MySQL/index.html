DDL

  • Data Definition Language 数据定义语言，用来定义数据库对象；数据库，表，列等
  • 操作数据库
    • 登录: mysql -u用户名
    • p密码
    • 查看所有库：show databases;
    • 进入库：use 库名;
    • 创建：create database 库名;
    • 删除：drop database 库名;
    • 展示库中所有的表：show tables;
    • 查看当前使用的库：select database();

操作数据表

  • 创建表

    -> create table tb1(

      -> id int,
  -> name varchar(16)               varchar->字符串 
  -> )default charset=utf8;

    -> create table tb2(

      -> id int,
  -> name varchar(16) not null,     不允许为空
  -> email varchar(32) null,        允许为空（默认）
  -> age int
  -> )default charset=utf8;

    -> create table tb3(

      -> id int,
  -> name varchar(16) not null,
  -> email varchar(32) null,
  -> age int default 3              插入数据时如果不设置值默认为3
  -> )default charset=utf8;

    主键：不允许为空，不能重复，用于表示这条数据的id编号，一般与自增一起用
    注意：一个表只能有一个自增列
    -> create table tb4(

      -> id int not null auto_increment primary key,     主键
  -> name varchar(16) not null,
  -> email varchar(32) null,
  -> age int default 3              插入数据时如果不设置值默认为3
  -> )default charset=utf8;

  • 删除表：drop table 表名;

  • 清楚表：delete from 表名;

  • 查看表结构：desc 表名;

  • 修改表

    • 增加一列：alter table tb4 add password varchar(64);

  • 查看表内容：select * from 表名


DML

  • Data Manipulation Language 数据操作语言，是指在SQL语言中，负责对数据库对象运行数据访问工作的指令集。
  • 增加数据
    • insert into 表名（列名1，列名2，…）（哪一列的数据） values(值1，值2)（插入的值）;
  • 删除数据 (不加条件，所有数据都会修改！)
    • delete from 表名 [where 条件] (通过条件找到该条数据);
  • 修改数据 (不加条件，所有数据都会修改！)
    • update 表名 set 列名1=值1，列名2=值2，(需要更新成为的值)…[where 条件] (通过条件找到该条数据);

DQL

  • Data QueryLanguage 数据查询

  • 基础查询

    • select 列1,列2,··· from 表名
      只查询这几列
    • select * from 表名
      查询所有列

  • 过滤查询
    select * from 表名 where 条件

  • 聚合分组

    • group by出现哪一列，哪一列才能出现在select的非聚合中
    • 示例：
      select sex, avg(age), min(age), max(age), count(*) from tb1 group by sex(性别)

  • 排序分页

    • limit: n,m 跳过前面n条再取m条
    • order by: asc：升序 desc：降序
